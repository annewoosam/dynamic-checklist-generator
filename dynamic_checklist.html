Creator Questions

Hidden Question Elements that appear at Preparer and Reviewer Stage

Preparer Questions

The preparer should be able to click a button and generate the checklist for all questions, as well as three additional input fields for each question: prepStatus(a drop down with a blank default), prepComments(a text area) and prepMinutes (a number field). A send to reviewer button that logs date and time clicked should also be implemented.

Reviewer Questions

The reviewer should be able to click a button and generate the checklist for all questions, as well as three additional input fields for each question: reviewStatus(a drop down with a blank default), reviewComments(a text area) and reviewMinutes (a number field). Two buttons, one to Return for Corrections and another to Send to Recipient should also be implemented.
